# 💸 Personal Finance Dashboard

A modern and interactive **Personal Finance Dashboard** built using **HTML, CSS, and JavaScript**.
This project helps users track transactions, visualize spending, and gain financial insights.

---

## 🚀 Features

### 📊 Dashboard Overview

* Displays **Total Balance, Income, and Expenses**
* **Balance Trend Chart** (time-based visualization)
* **Category Breakdown Chart** (spending distribution)

---

### 📄 Transactions Section

* View all transactions with:

  * Date
  * Category
  * Amount
  * Type (Income / Expense)
* Features:

  * 🔍 Search by category
  * 🎯 Filter by type
  * 🔄 Reset filters

---

### 👥 Role-Based UI (Frontend Simulation)

* **Viewer**

  * Can only view data
* **Admin**

  * Can add transactions
  * Can delete transactions

Switch roles using the dropdown to see UI changes.

---

### 📈 Insights Section

* Shows:

  * 🥇 Highest spending category
  * 💰 Total transaction amount
  * 📊 Income vs Expense insight
  * 📌 Smart message based on spending behavior

---

### 🧠 State Management

* Managed using a centralized `state` object
* Tracks:

  * Transactions
  * Filters
  * User role

---

### 💾 Data Persistence

* Uses **Local Storage**
* Transactions remain saved even after page refresh

---

## 🎨 UI & UX

* Clean and modern **glassmorphism design**
* Responsive layout (works on mobile & desktop)
* Smooth hover effects and transitions
* Handles empty states gracefully

---

## ⚙️ Tech Stack

* HTML5
* CSS3 (Flexbox + Glass UI)
* Vanilla JavaScript (No frameworks)
